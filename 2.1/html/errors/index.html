Номера ошибок в NTphone 2.1

  • #000: Пользователь вышел из телефона
  • #001: Неверная команда
  • #002: Деление на ноль
  • #003: Приложения
    • В папке не найдено приложений
    • Приложение не найдено
  • #004: Отказано в доступе на изменение настроек
  • #005: Переменные
    • Ошибка в типе данных
    • Ошибка в значениях переменной
  • #006: Не найдено
    • Файлы не найдены (ModuleNotFound)
    • Модули не найдены (FileNotFound)
    • Не установлены необходимые модули (Import)
    • Переменная\функция\класс\метод\аргумент не найден (Ошибка в коде телефона)
  • #007: Ошибка в коде телефона
    • Ошибка в ключе для словаря
    • Ошибка в синтаксе
  • #008: Ошибка при чтении файлов
    • Ошибка при чтении файлов (UnicodeDecode)
    • Ошибка при чтении файлов (UnicodeEncode)
  • #009: Закончилась память
  • #010: Предел информации
    • Слишком большой вывод арифметической операции
    • Предел ввода информации
    • Индекс в последовательности вышел из допустимого диапазона
  • #011: Ошибка в табуляции
    • Ошибка в табуляции (Tab) (Ошибка в коде телефона)
    • Ошибка в табуляции (Indenation) (Ошибка в коде телефона)
  • #012: Сбой системы
